# RGAN
Recurrent Adversarial Learning for Geotechnical Time-Series Augmentation: Application to Slope instability Forecasting in Open-Pit Mine

### Prerequisites

You will need Python 3.6+ and the following packages:

- Pandas  - 1.1.3
- Numpy - 1.21.2
- TensorFlow - 1.14.0
- Keras - 2.2.5
- Matplotlib - 3.3.3
- Scikit-learn - 0.23.2
